Вам необходимо доработать игру — гонки. Игрок-пользователь соревнуется с противником, сгенерированным компьютером. Победитель определяется по сравнению характеристик. Действуйте по пунктам:
1. Внимательно прочитайте код. Разберитесь, как он устроен.
2. Допечатайте все нужные характеристики — для этого нужно обратиться к полям класса.
3. Реализуйте методы:
4. Параметры авто игрока будут подгружаться из отдельного сервиса, хранение данных в котором отличается от используемого в игре. Метод `createCarByProperties(CarProperties carProperties)`, должен конвертировать параметры в нужные типы и возвращает экземпляр класса `Car`.
5. Метод `Integer getNitroLevel(int nitroLevel)` должен возвращать значение `nitroLevel`. Если `nitroLevel` равен нулю — тогда нужно вернуть `null`.
6. Метод `makeRace(Car userCar, Car opponentCar, int distance)` должен определять победителя и возвращать количество полученных или проигранных очков:
   * Если гоночная дистанция — переменная `distance` — меньше или равна 15 километрам, то победителем объявляется тот игрок, у автомобиля которого больше ускорение — поле `acceleration`. <br> Если дистанция больше 50 километров, то выигрывает тот, у автомобиля которого наибольшая максимальная скорость — поле `maxSpeed`.
   * В случае победы игрока метод возвращает количество очков, равное наибольшему из максимальных скоростей участников.
   * Если по первому условию определить победителя не удалось, сравниваются ускорения игроков. Если они равны, объявляется ничья и метод возвращает `0` очков.
   * Если и ничью определить не удалось, сравнивается уровень закиси азота. Если он строго больше, то метод возвращает `0` очков. Если нет, метод возвращает `-100` очков.
7. Метод `changePointAndDistance(...)` должен менять количество очков на переданный аргумент `points` и увеличивать количество пройденных километров на аргумент `distance`.

### Класс Car
```java
public class Car {
    double maxSpeed; // максимальная скорость
    float acceleration; // ускорение
    int score; // количество очков
    double kilometersTravelled; // количество пройденных километров
    Integer nitroLevel; // уровень закиси азота

    public Car(double maxSpeed, float acceleration, int score, Integer nitroLevel) {
        this.maxSpeed = maxSpeed;
        this.acceleration = acceleration;
        this.score = score;
        this.nitroLevel = nitroLevel;
    }
}
```

#### Подсказки:
* Чтобы напечатать характеристики автомобилей нужно обратиться к полям `maxSpeed`, `acceleration` и `nitroLevel` через имя объекта `userCar` или `opponentCar` и точку.
* В методе `createCarByProperties(CarProperties carProperties)` для конвертации `maxSpeed` из типа `String` в примитив `double` используйте метод `parseDouble`. Для конвертации `acceleration` из `double` во `float` необходимо использовать явное приведение `(float)`. Параметр `score` из `byte` в `int` Java приведёт автоматически.
* Для конвертации `nitroLevel` необходимо вызвать метод `getNitroLevel(int nitroLevel)` и передать ему в качестве аргумента `carProperties.nitroLevel`.
* Условие победы в методе `makeRace(Car userCar, Car opponentCar, int distance)`: или `(||)` одновременно выполняются условия `(distance <= 15)` и `(&&)` `(userCar.acceleration > opponentCar.acceleration)`, или `(distance > 50)` и `(&&)` `(userCar.maxSpeed > opponentCar.maxSpeed)`.
* В методе `makeRace(Car userCar, Car opponentCar, int distance)` найти максимум двух чисел типа `double` можно с помощью метода `Double.max()`. Метод `Double.max()` возвращает тип `double`, который для корректной работы метода нужно явно привести к `int`.
* Чтобы была ничья, должны совпасть ускорения `userCar.acceleration == opponentCar.acceleration`.
* Чтобы изменить параметры у автомобиля пользователя в методе `changePointAndDistance()`, необходимо передать экземпляр класса `Car`, очки `points` и дистанцию `distance` — `changePointAndDistance(Car car, int points, int distance)`.
* Чтобы посчитать очки и километры используйте операции `car.score += points;` и `car.kilometersTravelled += distance;`.
